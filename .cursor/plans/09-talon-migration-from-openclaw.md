
# Talon — Migration Guide: OpenClaw → Talon

## What We're Moving

| OpenClaw Asset | Location | Talon Destination |
|---|---|---|
| Core memories | `MEMORY.md`, `USER.md`, `SOUL.md` | `data/memories/*.md` → `core_matrix.json` |
| Daily logs | `memory/YYYY-MM-DD.md` | PostgreSQL episodic table (SQL import) |
| Skills | `~/.openclaw/skills/<name>/index.ts` | `backend/skills/<name>/main.py` + `skill.toml` |
| Config & credentials | `~/.openclaw/openclaw.json`, `credentials/` | `config/talon.toml` + `config/secrets/` |

## Migration Order

```bash
python scripts/migrate_memories.py \
  --openclaw-workspace ~/.openclaw --talon-root /root/talon

psql -U talon -d talon -f scripts/episodic_import.sql

python scripts/migrate_skills.py \
  --openclaw-skills ~/.openclaw/skills --talon-skills /root/talon/backend/skills

python scripts/migrate_config.py \
  --openclaw-config ~/.openclaw/openclaw.json --talon-config /root/talon/config

python scripts/validate_migration.py --talon-root /root/talon
```

Add `--dry-run` to any script to preview without writing files.

## Skill Porting Effort

| Skill | Complexity | Notes |
|---|---|---|
| `searxng-enhanced` | Low | HTTP GET to localhost:8080, parse JSON |
| `yahoo-finance` | Low | httpx to Yahoo Finance, same pattern |
| `weather-enhanced` | Low | HTTP call + response reshape |
| `hostinger-email` | Medium | SMTP or REST — check auth in index.ts |
| `news-sentinel` | Medium | Depends on news source query pattern |
| `bird` | Low–Medium | Port API calls if still in use |
| `foundry-openclaw` | N/A | Replaced by Talon's Skills Engine |

## What You Don't Migrate

| OpenClaw Component | Talon Equivalent | Action |
|---|---|---|
| `memory-core` plugin | Built-in Memory Engine | No migration |
| `system-guardian` plugin | Sentinel + logging | No migration |
| `sessions` tool | ChatRouter built-in | No migration |
| `openclaw-gateway.service` | `talon.service` | Replace entirely |
| `openclaw.json` | `config/talon.toml` | Regenerated by migrate_config.py |
